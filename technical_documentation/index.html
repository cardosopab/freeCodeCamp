Introduction

Vim is a powerful command-line code editor tool that is an enhanced version of the original vi editor. It is available for most operating systems, including macOS, Windows, and Linux. Vim is a modal editor, which means it has different modes, such as command mode and insert mode, that determine the function of the keys on your keyboard. Vim is popular among programmers, coders, and system administrators due to its efficiency and the ability to work with multiple documents and windows.

History

Vim was created in 1991 as a free, open-source software. It was designed as an improved version of the vi editor, which was created for Unix in the 1970s. Vim stands for "vi improved" and was initially designed with Unix in mind, but now has versions for most operating systems, including Android and iOS smartphones.

Prior knowledge

Before diving into Vim, it's helpful to be familiar with basic text editing concepts and keyboard shortcuts, as Vim relies on unique commands and modes to edit text efficiently. If you're new to Vim, you can run vimtutor in the terminal to learn some initial commands and get started.

Basic
  • :w

    Save file

  • :q

    Quit Vim

  • :q!

    Quit Vim without saving

  • :help [keyword]

    Perform a search of help documentation for the specified keyword

Movement
  • h

    Move left

  • j

    Move down

  • k

    Move up

  • l

    Move right

  • w

    Move to beginning of word

  • e

    Move to end of word

Insertion
  • i

    Insert before cursor

  • a

    Insert after cursor

  • o

    Insert into a newly created line under cursor

  • I

    Insert at the beginning of the line

  • A

    Insert at the end of the line

  • O

    Insert into a newly created line above cursor

  • Esc

    Return to command mode

Editing
  • y

    Copy (yank) the highlighted section

  • yy

    Copy (yank) the current line

  • p

    Paste

    • Flow

  • u

    Undo last change

  • Ctrl + r

    Redo last change

    • Delete

  • dd

    Deletes the line cursor is on

  • x

    Delete character under cursor

    • Alter

  • ctrl + a

    Increment number

  • ctrl + x

    Decrement number

  • r

    Replace character under cursor

  • ciw

    Change in word (deletes word and goes into insert mode)

  • ci+"character"

    Change in character (deletes everything within matching characters e.g. parenthesis/brackets and goes into insert mode)

  • ca+"character"

    Change outside character (deletes everything including matching characters e.g. parenthesis/brackets and goes into insert mode)

    • Find

  • /:

    Search for a string in the current file

  • n

    Move to the next search result

  • N

    Move to the previous search result

  • :s/"old characters"/"new characters"/g

    Replace old characters with new ones in the whole line (Press Enter to commit change, Esc to decline)

    • Highlight

  • v

    Enter visual mode and highlight spot

  • V

    Enter visual mode and highlight line

  • ctrl + v

    Enter visual mode and highlight spot and column depending on if you go up or down

Combinations
  • v + w

    Highlight words that can be deleted with x or copied with y

  • ctrl + v

    Highlight spot, then j or k to go down or up, adding to the characters highlighted in the column, and finally I or A to insert before or after the column for multi-line insertion
